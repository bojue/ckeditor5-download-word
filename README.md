@ckeditor/ckeditor5-export-word
===============================

ckeditor5的Word下载插件

### `start`

```bash
# Starts the server and open the browser.
yarn run start

# Disable auto-opening the browser.
yarn run start --no-open

# Create the editor with the interface in German.
yarn run start --language=de
```
